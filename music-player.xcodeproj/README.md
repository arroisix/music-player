# Music Player

A modern music player application built for macOS using SwiftUI.

## Features

- Music playback control
- Music library management
- Modern macOS design with hidden title bar
- Responsive UI optimized for 1400x900 window size

## Requirements

- macOS (compatible with SwiftUI)
- Xcode 15.0 or later
- Swift 5.9 or later

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/music-player.git
   ```

2. Open the project in Xcode:
   ```bash
   cd music-player
   open music-player.xcodeproj
   ```

3. Build and run the project (âŒ˜R)

## Usage

Launch the application to access the music player interface with library management and playback controls.

## Project Structure

- `music_playerApp.swift` - Main app entry point
- `MusicPlayerView.swift` - Main player interface
- `MusicLibraryView.swift` - Music library management
- `ContentView.swift` - Default content view
